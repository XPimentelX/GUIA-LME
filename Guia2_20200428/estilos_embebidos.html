Estilos embebidos en las etiquetas HTML

Usar estilos CSS embebidos en las etiquetas HTML es el peor y el menos utilizado de los métodos para agregar reglas de estilo. No es nda recomendable por la dificultad que implica realizar un cambio posterior.

Esta dorma de incluir CSS directamente en los elemenentos HTML solamente se utiliza en determinadas situaciones en las que se debe incluir un estilo muy especifico para un sólo elemento en concreto.

Para insertar un estilo CSS en un elemento se utiliza la propiedad style de HTML, que en ningun momento se debe confundir con la etiqueta < style>que se introduce en la cabecera del documento.
